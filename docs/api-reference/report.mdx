---
title: Get Full Report
api: GET /report/{id}
description: Retrieve the full report card (XRPL paywalled)
---

## Path Parameters

<ParamField path="id" type="string" required>
  Report ID returned from `POST /analyze`.
</ParamField>

## Authentication

This endpoint is behind an XRPL micropayment paywall (0.05 XRP) when `FOUNDER_SEED` is configured. If `FOUNDER_SEED` is not set, the endpoint is free.

### Payment Flow

1. Send at least 0.05 XRP to the founder wallet address on the configured XRPL network
2. Include the transaction hash in the `X-Payment-TxHash` request header
3. Optionally include `X-Payment-Sender` for sender verification (required if `PAYWALL_REQUIRE_SENDER` is set)

The paywall verifies `delivered_amount`, checks the payment is less than 24 hours old, and prevents tx hash reuse across different reports.

## Response

Returns the full `ReportCard` object with all analysis data.

        <ResponseField name="forks" type="number">Fork count.</ResponseField>
        <ResponseField name="openIssues" type="number">Open issue count.</ResponseField>

        <ResponseField name="totalCommits" type="number">Total commit count.</ResponseField>
        <ResponseField name="recentCommits" type="CommitSummary[]">Last N commits with SHA, message, date, and author.</ResponseField>
        <ResponseField name="contributors" type="ContributorSummary[]">Contributors with login and contribution count.</ResponseField>
        <ResponseField name="createdAt" type="string">Repo creation date (ISO 8601).</ResponseField>
        <ResponseField name="updatedAt" type="string">Last update date (ISO 8601).</ResponseField>
        <ResponseField name="description" type="string | null">Repo description.</ResponseField>

        <ResponseField name="hasCi" type="boolean">Whether repo has CI configuration.</ResponseField>
      </Expandable>

    <ResponseField name="completedAt" type="string | null">Analysis completion timestamp.</ResponseField>
    <ResponseField name="error" type="string | null">Error message if analysis failed.</ResponseField>
  </Expandable>
</ResponseField>
